# LocalHire

AI-powered recruitment platform built using ASP.NET Core 8, React and Google Gemini AI.

🌐 Live Demo- https://www.getlocalhire.com

## Features

- Candidate Registration
- Employer Registration
- JWT Authentication
- Email Verification
- Mobile Verification (MSG91)
- Resume Upload
- AI Resume Matching
- AI Candidate Ranking
- Job Posting
- Job Applications
- Employer Dashboard
- Candidate Dashboard
- Azure Deployment
- GitHub Actions CI/CD
